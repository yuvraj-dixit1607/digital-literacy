This repository is my Digital Literacy Portfolio, submitted as part of the CSE0001 course project at VIT Bhopal University. As a Student Digital Ambassador, I completed five tasks covering the core pillars of digital literacy — designing awareness infographics, building a professional online presence, exploring coding platforms, practising email etiquette, and researching cybercrime awareness.

Module 5 | Topic: Phishing
